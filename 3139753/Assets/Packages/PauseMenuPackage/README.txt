===Simple Pause Menu Unity Package===


---Setup---
1. Open the scene you wish to have your pause menu in
2. Place the PauseMenuPackage into the scene
3. OPTIONAL - Inside the PauseMenuUI script change the Stage Select scene value
to that of your main menu
---Activate---
   - Press ESC while the PauseMenuPackage is in the scene to open the pause menu
   - Press the reset button to restart the current scene
   - Press Stage Select to change the scene to that of your main menu
   - Press either ESC or Return to close the pause menu
